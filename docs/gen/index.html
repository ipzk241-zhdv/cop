Tic-Tac-Toe Game

An advanced, highly customizable web version of the classic Tic-Tac-Toe game. This project supports dynamic board sizes, adjustable winning conditions and match history tracking.

Getting Started

To get a local copy up and running, ensure you have Node.js installed on your machine. Clone the repository, navigate to the project root directory, and install the required dependencies using the following command:

npm install

Basic Commands

Here are the primary scripts you can use to interact with the project environment:

  • npm run dev Start Dev Server - Starts the local development server with Hot Module Replacement (HMR).
  • npm run build Build Project - Compiles and bundles the application for production into the dist folder.
  • npm run preview Preview Build - Boots up a local static web server that serves the optimized files from dist.
  • npm run storybook Run Storybook - Opens the Storybook testing environment for interactive UI components.
  • npm run gen-docs Generate Docs - Builds the JSDoc HTML documentation from your source code comments.
